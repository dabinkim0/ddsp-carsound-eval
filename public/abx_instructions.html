Car Sound Synthesis Listening Test
Quick Guide

  • Listen to the Reference sample first and rate each candidate between 0 and 100 using the slider based on its similarity to the Reference.
    • 100 - Same as the Reference
    • 80 - Very similar and barely noticable differences
    • 60 - Noticeable differences but still similar
    • 40 - Significant differences
    • 20 - Clearly different
    • 0 - Completely different
  • You may listen to the Reference and each sample as many times as needed.
  • The order of candidates is randomized.
  • Note A: Please use earphones or headphones.
  • Note B: Please use the entire slider range from 0 to 100.
  • Note C: Please adjust the volume to a comfortable level before starting. During the test, try to keep the listening environment and volume consistent.
  • 먼저 Reference 샘플을 듣고, 각 후보 샘플을 Reference와의 유사도를 기준으로 0부터 100까지 슬라이더로 평가하세요.
    • 100 - Reference와 동일함
    • 80 - 미세한 차이가 존재하지만 매우 유사함
    • 60 - 약간의 차이가 존재하지만 유사함
    • 40 - 유의미한 차이가 있음
    • 20 - 명백히 다름
    • 0 - 전혀 다름
  • Reference 및 각 샘플은 필요한만큼 반복해서 들어도 됩니다.
  • 각 문항의 샘플 순서는 랜덤으로 섞여 있으며, Reference를 찾으려 하지 말고 유사도를 기준으로 평가하세요.
  • 주의사항 A: 이어폰 또는 헤드폰을 사용해주세요.
  • 주의사항 B: 0-100 전체 구간을 활용해주세요.
  • 주의사항 C: 테스트 시작 전에 편안하게 들을 수 있는 수준으로 볼륨을 조절해 주세요. 테스트 진행 중에는 가급적 청음 환경과 볼륨을 유지해 주세요.
